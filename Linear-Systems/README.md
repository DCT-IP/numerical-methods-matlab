# Linear Systems

This folder studies numerical methods for solving linear systems of the form

    A x = b

and related least-squares problems.

## Primary Source
- Cleve Moler, *Numerical Computing with MATLAB*
  - Linear Equations
  - Least Squares

## Topics Covered
- Floating-point effects and numerical error
- Gaussian elimination and LU factorization
- The backslash operator (`A\b`)
- Overdetermined systems and least squares
- Conditioning and numerical stability
